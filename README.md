📝 To-Do List Web App
A simple and interactive To-Do List application built using HTML, CSS, and JavaScript.
This project demonstrates fundamental front-end development concepts such as DOM manipulation, event handling, and browser storage.

🚀 Project Overview
This To-Do List allows users to:

Add tasks
Delete tasks
Press Enter key to add tasks quickly
Save tasks in the browser using localStorage
Keep tasks even after refreshing the page
The goal of this project is to understand how JavaScript interacts with HTML elements and how data can be stored on the client side.

🛠 Technologies Used
Technology	Purpose
HTML	Structure of the web page
CSS	Styling and layout
JavaScript	App logic and interactivity
localStorage	Saving tasks in browser memory
⚙️ How the App Works
The user types a task into the input field.
When the + button or Enter key is pressed:
A new task object is created.
The task is displayed on the screen.
The task is saved in localStorage.
When the delete button is clicked:
The task is removed from the list.
The browser storage updates automatically.
